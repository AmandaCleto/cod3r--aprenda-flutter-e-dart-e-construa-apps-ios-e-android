### Installing dependencies:
Run the command to install the project dependencies.
   ```sh
   flutter pub get
   ```
<br>

   ```sh
   flutter create .
   ```

### Running the project:
The scripts below run the project.
   ```sh
   flutter run
   ```

### 👍 Conclusion: 

After following the previous steps, the project is running in **development mode**.
